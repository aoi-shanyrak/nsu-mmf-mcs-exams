## Материалы для подготовки к экзаменам (Мехмат НГУ)

Репозиторий содержит структурированные материалы для подготовки к экзаменам и зачётам на механико-математическом факультете НГУ.
## Цель проекта

Создание открытой базы знаний с конспектами, определениями, теоремами и практическими материалами для студентов мехмата НГУ.
## Содержание

Материалы организованы по семестрам и предметам. Включают:

    Определения и формулировки теорем
    Списки обязательных теорем (без доказательств)
    Официальные конспекты в формате pdf

 ## Как использовать

    Найдите нужный предмет в соответствующей сессии

    Используйте материалы для повторения теории

    Дополняйте своими конспектами

> скачивайте репозиторий и смотрите в obsidian. там latex формулы отображаются корректно 


## Вклад в проект

Приветствуются любые улучшения:

    Добавление новых материалов

    Исправление ошибок и неточностей

    Улучшение структуры и оформления

Материалы носят неофициальный характер и предназначены для помощи в обучении. Рекомендуется сверяться с официальными источниками и лекционными материалами.
