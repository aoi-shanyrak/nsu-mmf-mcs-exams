
### **Топологическое определение предела функции**  
Пусть D ⊆ ℝ, f: D → ℝ, x₀ — предельная точка множества D, y₀ ∈ ℝ̅ (расширенная числовая прямая). Тогда y₀ называется пределом функции f при x → x₀, если для любой окрестности V точки y₀ существует такая проколотая окрестность U точки x₀, что для всех x ∈ D ∩ U выполняется f(x) ∈ V.

### **Предел функции слева**  
Число y₀ называется пределом функции f слева в точке x₀ (обозначение: lim_{x→x₀⁻} f(x) = y₀), если для любого ε > 0 существует δ > 0 такое, что для всех x ∈ D, удовлетворяющих условию x₀ - δ < x < x₀, выполняется |f(x) - y₀| < ε.

### **Предел функции справа**  
Число y₀ называется пределом функции f справа в точке x₀ (обозначение: lim_{x→x₀⁺} f(x) = y₀), если для любого ε > 0 существует δ > 0 такое, что для всех x ∈ D, удовлетворяющих условию x₀ < x < x₀ + δ, выполняется |f(x) - y₀| < ε.

### **Односторонний предел функции**  
Пределы слева и справа называются односторонними пределами.

### **Двусторонний предел функции**  
Число y₀ называется двусторонним пределом функции f в точке x₀ (обозначение: lim_{x→x₀} f(x) = y₀), если для любого ε > 0 существует δ > 0 такое, что для всех x ∈ D, удовлетворяющих условию 0 < |x - x₀| < δ, выполняется |f(x) - y₀| < ε.

### **Связь односторонних и двустороннего пределов**  
Двусторонний предел функции f в точке x₀ существует тогда и только тогда, когда существуют оба односторонних предела в этой точке и они равны между собой. В этом случае:

lim_{x→x₀} f(x) = lim_{x→x₀⁻} f(x) = lim_{x→x₀⁺} f(x).
