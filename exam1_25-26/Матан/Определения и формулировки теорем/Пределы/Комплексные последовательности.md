
### **Координатный критерий сходимости последовательности**  
Пусть `z_n = x_n + i y_n` — последовательность комплексных чисел, а `c = a + i b`, где `a, b ∈ R`. Тогда утверждение
```
z_n стремится к c при n, стремящемся к бесконечности
```
равносильно одновременному выполнению двух условий:
```
x_n стремится к a  и  y_n стремится к b при n, стремящемся к бесконечности.
```

### **Полярный признак сходимости последовательности**  
Пусть последовательность задана в полярной форме: `z_n = r_n (cos θ_n + i sin θ_n)`, где `r_n = |z_n| ≥ 0` — модуль, а `θ_n = arg z_n` — аргумент числа `z_n`. Если выполняются два условия:
1. Модули стремятся к пределу: `r_n → r ≥ 0`,
2. Аргументы стремятся к пределу (по модулю `2π`): `θ_n → θ`,
то сама последовательность сходится:
`z_n → r (cos θ + i sin θ)`.

### **Экспонента мнимого аргумента (Формула Эйлера)**  
Для любого вещественного числа `y` справедлива знаменитая формула Эйлера, связывающая комплексную экспоненту с тригонометрическими функциями:
`exp(i y) = cos y + i sin y`.

### **Предел последовательности вида (1 + z_n / n)^n**  
Пусть дана комплексная последовательность `z_n = x_n + i y_n`, чьи действительная и мнимая части сходятся:
`x_n → x` и `y_n → y`.
Тогда справедливо следующее предельное соотношение, обобщающее второй замечательный предел на комплексный случай:
```
lim_{n→∞} (1 + (x_n + i y_n)/n)^n = exp(x + i y) = e^x * (cos y + i sin y).
```
