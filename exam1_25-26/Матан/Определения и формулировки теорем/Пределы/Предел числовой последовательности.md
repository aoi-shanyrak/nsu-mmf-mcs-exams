
### **Последовательность чисел**  
Последовательностью элементов множества A называется отображение a: ℕ → A. Для элемента последовательности вместо a(n) пишут aₙ. Последовательность обозначается {aₙ ∈ A}ₙ∈ℕ или просто {aₙ}ₙ.

### **Ограниченная последовательность чисел**  
Последовательность чисел {xₙ} называется ограниченной, если множество её значений ограничено, т.е. существует такое число C > 0, что для всех n ∈ ℕ выполнено |xₙ| ≤ C.

### **Неограниченная последовательность чисел**  
Последовательность называется неограниченной, если она не является ограниченной (для любого C > 0 найдётся номер n такой, что |xₙ| > C).

### **Конечный предел последовательности чисел**  
Число a ∈ ℝ называется (конечным) пределом последовательности {xₙ} при n → ∞, если для любого ε > 0 существует номер Nₑ ∈ ℕ такой, что для всех n ≥ Nₑ выполнено |xₙ - a| < ε. Обозначение: a = lim_{n→∞} xₙ.

### **Бесконечный предел последовательности чисел**  
Последовательность {xₙ} имеет бесконечный предел +∞ (-∞) при n → ∞, если для любого числа M ∈ ℝ существует номер N ∈ ℕ такой, что для всех n ≥ N выполнено xₙ > M (соответственно xₙ < M). Обозначение: lim_{n→∞} xₙ = +∞ (или -∞).

### **Топологическое определение предела последовательности**  
Значение a ∈ ℝ̅ (расширенной числовой прямой) является пределом последовательности чисел xₙ при n → ∞, если для любой окрестности U точки a существует номер Nᵤ ∈ ℕ такой, что для всех n ≥ Nᵤ выполнено xₙ ∈ U.

### **Сходящаяся последовательность чисел**  
Последовательность называется сходящейся, если у неё существует конечный предел.

### **Расходящаяся последовательность чисел**  
Последовательность называется расходящейся, если она не сходится (либо не имеет предела, либо имеет бесконечный предел).

### **Связь предела и арифметических операций**  
Если последовательности {xₙ} и {yₙ} сходятся, то:
1. Предел суммы равен сумме пределов: lim (xₙ + yₙ) = lim xₙ + lim yₙ.
2. Предел произведения равен произведению пределов: lim (xₙ · yₙ) = lim xₙ · lim yₙ.
3. Если lim yₙ ≠ 0, то предел частного равен частному пределов: lim (xₙ / yₙ) = (lim xₙ) / (lim yₙ).

### **Связь предела и неравенств**  
1. Если начиная с некоторого номера xₙ ≤ yₙ и обе последовательности сходятся, то lim xₙ ≤ lim yₙ.
2. Если lim xₙ < lim yₙ, то начиная с некоторого номера xₙ < yₙ.
3. (Теорема о промежуточной последовательности) Если начиная с некоторого номера xₙ ≤ yₙ ≤ zₙ и lim xₙ = lim zₙ = a, то lim yₙ = a.
