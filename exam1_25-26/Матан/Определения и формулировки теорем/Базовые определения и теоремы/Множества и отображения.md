
### **Образ множества под действием отображения**  
Пусть дано отображение `f: X -> Y` и множество `A`, являющееся подмножеством `X` (`A ⊆ X`). Образом множества `A` под действием отображения `f` называется множество всех элементов из `Y`, которые являются образами элементов из `A`. Обозначается `f(A)`:
`f(A) = { f(x) ∈ Y | x ∈ A }`.

### **Множество уровня функции**  
Пусть дана функция `f: X -> R`. Для любого числа `c ∈ R` множество уровня (множество уровня `c`) — это множество всех аргументов, на которых функция принимает значение `c`:
`{ x ∈ X | f(x) = c }`.

### **Прообраз множества под действием отображения**  
Пусть дано отображение `f: X -> Y` и множество `B`, являющееся подмножеством `Y` (`B ⊆ Y`). Прообразом множества `B` называется множество всех элементов из `X`, образы которых принадлежат `B`. Обозначается `f^{-1}(B)`:
`f^{-1}(B) = { x ∈ X | f(x) ∈ B }`.

### **Инъекция (однозначное отображение)**  
Отображение `f: X -> Y` называется инъективным (инъекцией), если разные элементы из `X` переходят в разные элементы из `Y`:
`∀ x1, x2 ∈ X, из x1 ≠ x2 следует f(x1) ≠ f(x2)`.
Эквивалентное условие: если `f(x1) = f(x2)`, то обязательно `x1 = x2`.

### **Сюръекция (отображение на)**  
Отображение `f: X -> Y` называется сюръективным (сюръекцией, отображением на `Y`), если каждый элемент множества `Y` является образом хотя бы одного элемента из `X`:
`∀ y ∈ Y существует x ∈ X: f(x) = y`.
То есть `f(X) = Y`.

### **Биекция (взаимно однозначное отображение)**  
Отображение `f: X -> Y` называется биективным (биекцией), если оно одновременно является инъективным и сюръективным.

### **Обратная функция**  
Пусть `f: X -> Y` — биективное отображение. Функция `g: Y -> X` называется обратной к `f`, если для любого `x ∈ X` верно `g(f(x)) = x` и для любого `y ∈ Y` верно `f(g(y)) = y`. Обозначается `f^{-1}`.

### **Обратимая функция**  
Функция `f: X -> Y` называется обратимой, если существует обратная функция `f^{-1}: Y -> X`.

### **Критерий обратимости функции**  
Функция `f: X -> Y` обратима тогда и только тогда, когда она является биекцией (то есть взаимно однозначным отображением между `X` и `Y`).

### **Композиция функций**  
Пусть даны две функции `f: X -> Y` и `g: Y -> Z`. Композицией (суперпозицией) функций `f` и `g` называется функция `h: X -> Z`, определяемая равенством `h(x) = g(f(x))` для всех `x ∈ X`. Обозначается `g ∘ f`.

### **Монотонная функция**  
Функция `f: R -> R` называется:
*   монотонно возрастающей, если для любых `x1, x2 ∈ R` из `x1 < x2` следует `f(x1) ≤ f(x2)`;
*   монотонно убывающей, если для любых `x1, x2 ∈ R` из `x1 < x2` следует `f(x1) ≥ f(x2)`.
Если неравенства строгие (`<` или `>`), то функция называется строго возрастающей или строго убывающей соответственно.

### **Четная функция**  
Функция `f: R -> R` называется четной, если для любого `x ∈ R` верно равенство `f(-x) = f(x)`. График четной функции симметричен относительно оси ординат (оси OY).

### **Нечетная функция**  
Функция `f: R -> R` называется нечетной, если для любого `x ∈ R` верно равенство `f(-x) = -f(x)`. График нечетной функции симметричен относительно начала координат.

### **Выпуклая функция**  
Функция `f: R -> R` называется выпуклой вниз на интервале `(a, b)`, если для любых точек `x1, x2 ∈ (a, b)` и любого числа `λ` из отрезка `[0, 1]` выполняется неравенство Йенсена:
`f(λ*x1 + (1-λ)*x2) ≤ λ*f(x1) + (1-λ)*f(x2)`.
Геометрически это означает, что график функции на этом интервале лежит ниже хорды, соединяющей любые две его точки.

### **Строго выпуклая функция**  
Функция `f: R -> R` называется строго выпуклой вниз на интервале `(a, b)`, если для любых различных точек `x1, x2 ∈ (a, b)` и любого `λ ∈ (0, 1)` выполняется строгое неравенство:
`f(λ*x1 + (1-λ)*x2) < λ*f(x1) + (1-λ)*f(x2)`.

### **Вогнутая функция**  
Функция `f: R -> R` называется вогнутой (или выпуклой вверх) на интервале `(a, b)`, если для любых `x1, x2 ∈ (a, b)` и любого `λ ∈ [0, 1]` выполняется неравенство:
`f(λ*x1 + (1-λ)*x2) ≥ λ*f(x1) + (1-λ)*f(x2)`.
Геометрически график функции лежит выше хорды.

### **Строго вогнутая функция**  
Функция `f: R -> R` называется строго вогнутой на интервале `(a, b)`, если для любых различных `x1, x2 ∈ (a, b)` и любого `λ ∈ (0, 1)` выполняется строгое неравенство:
`f(λ*x1 + (1-λ)*x2) > λ*f(x1) + (1-λ)*f(x2)`.
