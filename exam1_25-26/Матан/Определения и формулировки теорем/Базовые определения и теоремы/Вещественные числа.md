
### **Промежуток. Интервал, отрезок, луч, числовая прямая**  
Числовая прямая — это множество всех вещественных чисел ℝ. Промежуток — это любое связное подмножество числовой прямой. Основные виды промежутков:
- Интервал (a, b) = { x ∈ ℝ | a < x < b }
- Отрезок [a, b] = { x ∈ ℝ | a ≤ x ≤ b }
- Луч (a, +∞) = { x ∈ ℝ | x > a } или (-∞, b) = { x ∈ ℝ | x < b }
- Полуинтервал: [a, b) = { x ∈ ℝ | a ≤ x < b } или (a, b] = { x ∈ ℝ | a < x ≤ b }

### **Аксиома непрерывности**  
Если A, B ⊆ ℝ — непустые множества такие, что для всех a ∈ A и всех b ∈ B выполнено a ≤ b, то существует такое число c ∈ ℝ, что для всех a ∈ A и всех b ∈ B выполнено a ≤ c ≤ b.

### **Принцип Коши-Кантора вложенных отрезков**  
Для всякой последовательности вложенных отрезков числовой прямой [a₁, b₁] ⊇ [a₂, b₂] ⊇ ... ⊇ [aₙ, bₙ] ⊇ ... существует по крайней мере одно число c ∈ ℝ, принадлежащее всем этим отрезкам, то есть c ∈ ∩_{n=1}^{∞} [aₙ, bₙ].

### **Принцип Архимеда**  
Множество натуральных чисел неограничено сверху: для любого вещественного числа x существует натуральное число n такое, что n > x.

### **Целая часть числа**  
Целой частью вещественного числа x называется наибольшее целое число, не превосходящее x: [x] = max{ k ∈ ℤ | k ≤ x }.

### **Расширенная числовая прямая и порядок на ней**  
Расширенной числовой прямой называется множество ℝ̅ = ℝ ∪ {-∞, +∞}. Порядок на ℝ̅ продолжает порядок на ℝ с добавлением условий: для любого x ∈ ℝ выполняется -∞ < x < +∞.
