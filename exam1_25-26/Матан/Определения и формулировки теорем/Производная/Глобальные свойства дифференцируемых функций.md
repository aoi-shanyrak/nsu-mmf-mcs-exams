
### **Теорема Ролля о среднем**  
Пусть функция f удовлетворяет условиям:
1. Непрерывна на отрезке [a, b]
2. Дифференцируема на интервале (a, b)
3. f(a) = f(b)
Тогда существует точка c ∈ (a, b) такая, что f'(c) = 0.

### **Теорема Лагранжа о среднем (теорема о конечных приращениях)**  
Пусть функция f удовлетворяет условиям:
1. Непрерывна на отрезке [a, b]
2. Дифференцируема на интервале (a, b)
Тогда существует точка c ∈ (a, b) такая, что выполняется равенство:
f(b) - f(a) = f'(c)·(b - a).
Эта формула называется формулой конечных приращений.

### **Теорема о конечных приращениях (обобщение Лагранжа)**  
Часто теоремой о конечных приращениях называют именно теорему Лагранжа, представленную выше. В более общем контексте — это утверждение о существовании точки c между x₁ и x₂ такой, что f(x₂) - f(x₁) = f'(c)·(x₂ - x₁).

### **Дифференциальный признак монотонности**  
Пусть функция f непрерывна на промежутке I и дифференцируема внутри него.
1. Если f'(x) ≥ 0 для всех x внутри I, то f монотонно возрастает на I.
2. Если f'(x) ≤ 0 для всех x внутри I, то f монотонно убывает на I.
3. Если f'(x) > 0 для всех x внутри I, то f строго возрастает на I.
4. Если f'(x) < 0 для всех x внутри I, то f строго убывает на I.

### **Дифференциальный признак выпуклости**  
Пусть функция f дважды дифференцируема на интервале (a, b).
1. Если f''(x) ≥ 0 для всех x ∈ (a, b), то f выпукла вниз (просто "выпукла") на (a, b).
2. Если f''(x) ≤ 0 для всех x ∈ (a, b), то f выпукла вверх (вогнута) на (a, b).
3. Если неравенства строгие, то выпуклость/вогнутость также строгие.
