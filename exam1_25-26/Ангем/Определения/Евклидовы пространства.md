# **Евклидовы пространства, метризованные аффинные пространства**

## **1. Скалярное произведение**

**Определение:** **Скалярным произведением** на вещественном векторном пространстве $V$ называется функция $(\cdot, \cdot): V \times V \to \mathbb{R}$, удовлетворяющая следующим аксиомам для любых $\vec{x}, \vec{y}, \vec{z} \in V$ и любого $\lambda \in \mathbb{R}$:

1. **Симметричность:** $(\vec{x}, \vec{y}) = (\vec{y}, \vec{x})$
2. **Линейность по первому аргументу:** $(\lambda \vec{x} + \vec{y}, \vec{z}) = \lambda (\vec{x}, \vec{z}) + (\vec{y}, \vec{z})$
3. **Положительная определённость:** $(\vec{x}, \vec{x}) \geq 0$, причём $(\vec{x}, \vec{x}) = 0 \iff \vec{x} = \vec{0}$

Векторное пространство с заданным на нём скалярным произведением называется **евклидовым пространством**.

---
## **2. Свойства скалярного произведения**

1. **Линейность по второму аргументу:** следует из симметричности и линейности по первому аргументу
2. **Невырожденность:** если $(\vec{x}, \vec{y}) = 0$ для всех $\vec{y}$, то $\vec{x} = \vec{0}$
3. **Свойства нормы:**
   - $\|\vec{x}\| \geq 0$, причём $\|\vec{x}\| = 0 \iff \vec{x} = \vec{0}$
   - $\|\lambda \vec{x}\| = |\lambda| \cdot \|\vec{x}\|$
   - Неравенство треугольника: $\|\vec{x} + \vec{y}\| \leq \|\vec{x}\| + \|\vec{y}\|$

---

## **3. Ортогональность векторов**

**Определение:** Векторы $\vec{x}$ и $\vec{y}$ называются **ортогональными**, если $(\vec{x}, \vec{y}) = 0$. Обозначение: $\vec{x} \perp \vec{y}$.
**Свойства:**
1. Нулевой вектор ортогонален любому вектору
2. Если $\vec{x} \perp \vec{y}$, то $\|\vec{x} + \vec{y}\|^2 = \|\vec{x}\|^2 + \|\vec{y}\|^2$ (теорема Пифагора)
3. Система ненулевых попарно ортогональных векторов линейно независима

---
## **4. Ортогональное дополнение**

**Определение:** Пусть $L$ — подпространство евклидова пространства $V$. **Ортогональным дополнением** к $L$ называется множество
$$L^\perp = \{ \vec{x} \in V \mid (\vec{x}, \vec{y}) = 0 \text{ для всех } \vec{y} \in L \}$$
**Свойства:**
1. $L^\perp$ — подпространство в $V$
2. $L \cap L^\perp = \{\vec{0}\}$
3. $V = L \oplus L^\perp$ (в случае конечномерного $V$)
4. $\dim L + \dim L^\perp = \dim V$
