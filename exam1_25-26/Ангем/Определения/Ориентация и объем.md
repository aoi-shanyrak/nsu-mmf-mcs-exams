## Ориентация векторного пространства

Пусть V — вещественное векторное пространство размерности n
### **1. Базисы и их деформация:** 
Два базиса e = (e₁, ..., eₙ) и f = (f₁, ..., fₙ) пространства V называются **непрерывно деформируемыми** друг в друга, если существует семейство базисов e(t) = (e₁(t), ..., eₙ(t)), непрерывно зависящее от параметра t ∈ [0, 1], такое что e(0) = e и e(1) = f

### **2. Эквивалентные базисы:** 
Базисы e и f называются **эквивалентными** (одинаково ориентированными), если определитель матрицы перехода C от e к f положителен: det C > 0

### **3. Ориентация пространства:**
Классы эквивалентности базисов по отношению "det C > 0" называются **ориентациями** пространства V. Пространство с выбранной ориентацией называется **ориентированным**

## Объем n-мерного параллелепипеда

Пусть V — евклидово пространство (со скалярным произведением ⟨·,·⟩)

### **4. Определение параллелепипеда:** 
**Параллелепипедом**, натянутым на векторы a₁, a₂, ..., aₖ ∈ V, называется множество:  
P(a₁, ..., aₖ) = { λ₁a₁ + ... + λₖaₖ | 0 ≤ λᵢ ≤ 1 }
При k = n параллелепипед называется **n-мерным**

### **5. Объем параллелепипеда:**
**k-мерный объем** параллелепипеда, натянутого на векторы a₁, ..., aₖ, определяется формулой:  
volₖ(a₁, ..., aₖ) = √[det G(a₁, ..., aₖ)]

где G(a₁, ..., aₖ) — **матрица Грама** системы векторов с элементами gᵢⱼ = ⟨aᵢ, aⱼ⟩
**Простейшие свойства объема:**
1. **Неотрицательность:** volₖ ≥ 0, причем =0 тогда и только тогда, когда векторы линейно зависимы
2. **Однородность:** volₖ(a₁, ..., λaᵢ, ..., aₖ) = |λ|·volₖ(a₁, ..., aᵢ, ..., aₖ)
3. **Аддитивность:** При выполнении условий вырожденности volₖ(x+y, a₂, ..., aₖ) = volₖ(x, a₂, ..., aₖ) + volₖ(y, a₂, ..., aₖ)
4. **Инвариантность:** Объем не меняется при замене aᵢ на aᵢ + μaⱼ (j ≠ i)

## Векторное произведение

Определено в **ориентированном трехмерном** евклидовом пространстве
### **6. Определение:**
**Векторным произведением** вектора a на вектор b называется вектор c = a × b, определяемый условиями:
1. |c| = |a|·|b|·|sin φ| (φ — угол между a и b)
2. c ортогонален и a, и b
3. Тройка (a, b, c) — **правая** (положительно ориентированная)

**Свойства:**
- Антикоммутативность: a × b = -(b × a)
- Билинейность
- Тождество Лагранжа: |a × b|² + ⟨a, b⟩² = |a|²·|b|²
- В координатах в правом ортонормированном базисе:  
    Если a = (a₁, a₂, a₃), b = (b₁, b₂, b₃), то  
    a × b = (a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁)

## Смешанное произведение

Определено в **ориентированном трехмерном** евклидовом пространстве

### **7. Определение:**
**Смешанным произведением** векторов a, b, c называется число (a, b, c) = ⟨a × b, c⟩
**Свойства:**
1. **Геометрический смысл:** |(a, b, c)| = vol₃(a, b, c) — объем параллелепипеда  
    Знак положителен, если тройка (a, b, c) правая  
    Теорема
2. **Кососимметричность:** Перестановка любых двух векторов меняет знак
3. **Циклическая инвариантность:** (a, b, c) = (b, c, a) = (c, a, b)
4. **Связь с определителем:** В правом ортонормированном базисе  
    (a, b, c) = det[a₁ b₁ c₁; a₂ b₂ c₂; a₃ b₃ c₃]
5. **Критерий компланарности:** a, b, c компланарны ⇔ (a, b, c) = 0
