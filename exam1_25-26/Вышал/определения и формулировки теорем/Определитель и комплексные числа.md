
### **31. Диагональные матрицы и трансвекции**
*   **Диагональная матрица** — квадратная матрица, у которой все элементы вне главной диагонали равны нулю. Обозначается как `D = diag(α₁, α₂, …, αₙ)`.
*   **Трансвекция** `Tᵢⱼ(β)` — элементарная матрица, которая получается из единичной матрицы заменой нулевого элемента на пересечении i-й строки и j-го столбца на число `β` (где `i ≠ j`).
*   **Умножение на трансвекцию**:
    *   Умножение матрицы **слева** на `Tᵢⱼ(β)` равносильно прибавлению к её i-й строке j-й строки, умноженной на `β`.
    *   Умножение матрицы **справа** на `Tᵢⱼ(β)` равносильно прибавлению к её j-му столбцу i-го столбца, умноженного на `β`.
*   **Умножение на диагональную**:
    *   Умножение **слева** на `D = diag(α₁, …, αₙ)` умножает каждую строку i на соответствующий коэффициент `αᵢ`.
    *   Умножение **справа** на такую матрицу умножает каждый столбец j на коэффициент `αⱼ`.

### **32. Теорема о разложении матрицы**
Любую обратимою квадратную матрицу `A` над полем можно представить в виде произведения диагональной матрицы и конечного числа трансвекций:
`A = T₁·T₂·…·Tᵣ·D·Tᵣ₊₁·…·Tₛ`.

### **33. Определитель обратимости матрицы**
*   **Определение:** Определитель квадратной матрицы `A = (aᵢⱼ)` порядка `n` над полем `P` — это элемент поля, вычисляемый по формуле:
    `det A = Σ (sign σ · a₁,σ(₁) · a₂,σ(₂) · … · aₙ,σ(ₙ))`, где сумма берётся по всем подстановкам `σ` степени `n`.
*   **Для n=2:** `|a b; c d| = a·d - b·c`
*   **Для n=3 (правило Саррюса):**
    `|a₁₁ a₁₂ a₁₃; a₂₁ a₂₂ a₂₃; a₃₁ a₃₂ a₃₃| = a₁₁·a₂₂·a₃₃ + a₁₂·a₂₃·a₃₁ + a₁₃·a₂₁·a₃₂ - a₁₃·a₂₂·a₃₁ - a₁₂·a₂₁·a₃₃ - a₁₁·a₂₃·a₃₂`

### **34. Определитель полураспавшейся и транспонированной матриц**
*   **Полураспавшаяся матрица** — блочная матрица вида `C = [A *; 0 B]`, где `A` и `B` — квадратные блоки, а `0` — нулевой блок.
    **Свойство:** `det C = det A · det B`.
*   **Транспонированная матрица** `Aᵀ` получается из `A` заменой строк на столбцы.
    **Свойство:** `det Aᵀ = det A`.

### **35. Поведение определителя при перестановке строк**
При перестановке любых двух строк матрицы её определитель меняет знак на противоположный.
*   **Следствие:** Если две строки матрицы одинаковы, её определитель равен нулю.

### **36. Поведение определителя при элементарных преобразованиях строк**
1.  **Перестановка строк:** Определитель меняет знак.
2.  **Умножение строки на скаляр λ:** Определитель умножается на λ.
3.  **Прибавление к одной строке линейной комбинации других строк:** Определитель не меняется.
*   **Следствие:** Если строки линейно зависимы, определитель равен нулю.

### **37. Теорема о существовании обратной матрицы**
Матрица `A ∈ Mₙ(P)` обратима (имеет обратную `A⁻¹`) **тогда и только тогда**, когда её определитель отличен от нуля: `det A ≠ 0`.

### **38. Определитель произведения матриц**
Определитель произведения квадратных матриц одного порядка равен произведению их определителей:
`det(A · B) = det A · det B`.

### **39. Разложение определителя по строке**
Пусть `A = (aᵢⱼ)` — квадратная матрица, `Mᵢⱼ` — минор (определитель матрицы, полученной вычёркиванием i-й строки и j-го столбца), `Aᵢⱼ = (-1)ⁱ⁺ʲ · Mᵢⱼ` — алгебраическое дополнение.
**Формула разложения по i-й строке:**
`det A = aᵢ₁·Aᵢ₁ + aᵢ₂·Aᵢ₂ + … + aᵢₙ·Aᵢₙ`.

### **40. Применение определителей к системам линейных уравнений (Крамер)**
Для квадратной невырожденной системы `n` уравнений с `n` неизвестными (`det A ≠ 0`) решение существует, единственно и находится по формулам Крамера:
`xⱼ = dⱼ / d`, где `d = det A`, а `dⱼ` — определитель матрицы, полученной из `A` заменой j-го столбца на столбец свободных членов.

### **41. Применение определителей к вычислению обратной матрицы**
Если `det A ≠ 0`, то обратная матрица вычисляется по формуле:
`A⁻¹ = (1 / det A) · A*`, где `A*` — **присоединённая матрица**, составленная из алгебраических дополнений элементов `Aᵢⱼ`, но **транспонированно** (`(A*)ᵢⱼ = Aⱼᵢ`).
Для матрицы 2x2: `A⁻¹ = 1/(ad-bc) · [d -b; -c a]`.

### **42-43. Поле комплексных чисел: определение, существование и единственность**
*   **Определение:** Поле комплексных чисел `C` — это минимальное поле, содержащее поле вещественных чисел `R` и элемент `i` (мнимую единицу), удовлетворяющий условию `i² = -1`. Каждый элемент имеет вид `z = a + bi`, где `a,b ∈ R`.
*   **Существование** доказывается построением поля из матриц вида `[a b; -b a]`, изоморфного `C`.
*   **Единственность** (с точностью до изоморфизма): Любое поле, удовлетворяющее определению, изоморфно полю `C`.

### **44. Формула Муавра. Извлечение корня**
*   **Тригонометрическая форма:** `z = r(cos φ + i sin φ)`, где `r = |z|` — модуль, `φ = arg z` — аргумент.
*   **Формула Муавра:** Для любого целого `n`:
    `[r(cos φ + i sin φ)]ⁿ = rⁿ(cos nφ + i sin nφ)`.
*   **Извлечение корня n-й степени:** Число `z = r(cos φ + i sin φ)` имеет ровно `n` различных корней степени `n`:
    `zₖ = ⁿ√r ( cos((φ+2πk)/n) + i sin((φ+2πk)/n) )`, где `k = 0, 1, …, n-1`.

### **45. Общая линейная группа и её важнейшие подгруппы**
*   **Общая линейная группа** `GLₙ(P)` — группа всех обратимых (с определителем ≠ 0) квадратных матриц порядка `n` над полем `P` относительно умножения.
*   **Специальная линейная группа** `SLₙ(P)` — подгруппа `GLₙ(P)`, состоящая из матриц с определителем, равным 1.
*   **Ортогональная группа** `Oₙ(P)` — подгруппа `GLₙ(P)`, состоящая из матриц, удовлетворяющих условию `A·Aᵀ = E` (сохраняющих стандартное скалярное произведение).
*   **Унитарная группа** `Uₙ` — подгруппа `GLₙ(C)`, состоящая из комплексных матриц, удовлетворяющих условию `A·Āᵀ = E` (сохраняющих эрмитово скалярное произведение).
